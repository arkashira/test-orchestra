# Partner Targets
## Introduction
To enhance the value proposition of test-orchestra, we will integrate with key SaaS and API partners in the web and mobile app development ecosystem. This document outlines our partner integration roadmap, highlighting 7 strategic partners, their free-tier limits, integration effort, and the specific user jobs they solve.

## Partner Integration Roadmap

1. **CircleCI**
	* Free-tier limit: 1 user, 1 project, 1000 minutes/month
	* Integration effort: M (Medium)
	* Value-add: Streamlines continuous integration and continuous deployment (CI/CD) pipelines, solving the user job of automating testing and deployment workflows.
	* Affiliate/Revenue-share: Yes, CircleCI has a partner program with revenue sharing opportunities.
2. **GitHub**
	* Free-tier limit: Unlimited public repositories, 1000 automation minutes/month
	* Integration effort: S (Small)
	* Value-add: Enhances collaboration and version control, solving the user job of managing code changes and tracking testing history.
	* Affiliate/Revenue-share: Yes, GitHub has a developer program with revenue sharing opportunities.
3. **Sentry**
	* Free-tier limit: 1 project, 1000 events/day
	* Integration effort: M (Medium)
	* Value-add: Provides error tracking and monitoring, solving the user job of identifying and resolving issues in production environments.
	* Affiliate/Revenue-share: Yes, Sentry has a partner program with revenue sharing opportunities.
4. **AWS Device Farm**
	* Free-tier limit: 1000 minutes/month
	* Integration effort: L (Large)
	* Value-add: Offers cloud-based testing for mobile apps, solving the user job of ensuring app compatibility and performance across various devices.
	* Affiliate/Revenue-share: Yes, AWS has a partner program with revenue sharing opportunities.
5. **Google Cloud Test Lab**
	* Free-tier limit: 1000 minutes/month
	* Integration effort: L (Large)
	* Value-add: Provides cloud-based testing for mobile apps, solving the user job of ensuring app compatibility and performance across various devices.
	* Affiliate/Revenue-share: Yes, Google Cloud has a partner program with revenue sharing opportunities.
6. **Jenkins**
	* Free-tier limit: Unlimited
	* Integration effort: M (Medium)
	* Value-add: Enhances automation and CI/CD workflows, solving the user job of streamlining testing and deployment processes.
	* Affiliate/Revenue-share: No
7. **Bitbucket**
	* Free-tier limit: Unlimited public repositories, 500 build minutes/month
	* Integration effort: S (Small)
	* Value-add: Provides version control and collaboration, solving the user job of managing code changes and tracking testing history.
	* Affiliate/Revenue-share: Yes, Bitbucket has a partner program with revenue sharing opportunities.

## Prioritization
The prioritization of these partners is based on the following factors:

* Affiliate/Revenue-share opportunities
* Free-tier limits and pricing models
* Integration effort and technical complexity
* Value-add and the specific user jobs solved
* Market presence and industry recognition

By integrating with these strategic partners, test-orchestra can enhance its value proposition, streamline workflows, and provide a more comprehensive testing solution for web and mobile app developers.
